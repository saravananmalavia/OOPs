AddressBookApp
______________

It is a console based java application.
The prime objective is to explain the real time usage and implementation of oops concepts.
which has the following features.

1. create an Address Book with size N
2. to add a new Address in the Address Book
3. to remove an Address from the Address Book
4. to search an Address against the name
5. to update an Address with new details
6. to list all the Address in the Address Book


Concept  Used
--------------

1. control statements
2. looping statements
3. class & objects
4. encapsulation
5. this keyword
6. constructor
7. private & public access modifier
8. array of objects
7. packages
